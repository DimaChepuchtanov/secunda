# Проект обработки валютных курсов

## Описание проекта

Этот проект предназначен для автоматического получения данных о валютных курсах JPY/RUB и USD/RUB с Московской биржи (MOEX), обработки этих данных, расчета абсолютной разницы между курсами и генерации Excel-файла с результатами. После обработки результатов проект отправляет отчет по электронной почте.

Проект включает следующие основные функции:
- Получение данных о курсах валют с API MOEX.
- Обработка и объединение данных.
- Расчет разницы между курсами.
- Генерация Excel-файла с результатами.
- Отправка отчета по электронной почте.

## Используемые технологии

- **Python**: Основной язык программирования.
- **pandas**: Для обработки и анализа данных.
- **requests**: Для выполнения HTTP-запросов к API MOEX.
- **smtplib**: Для отправки электронной почты.
- **xlsxwriter**: Для генерации Excel-файлов (используется через pandas).

## Запуск проекта

1. Убедитесь, что у вас установлен Python (версия 3.6 или выше).

2. Установите необходимые зависимости:
   ```
   pip install pandas requests
   ```

3. Запустите проект:
   ```
   python main.py
   ```

После запуска проект выполнит следующие действия:
- Получит данные о курсах JPY/RUB и USD/RUB за последние 30 дней.
- Обработает данные и рассчитает разницу между курсами.
- Сгенерирует Excel-файл `result.xlsx` с результатами.
- Отправит отчет по электронной почте на указанный адрес.

**Примечание**: Для отправки электронной почты используются настройки Yandex.Mail. Убедитесь, что у вас есть доступ к учетной записи отправителя.
